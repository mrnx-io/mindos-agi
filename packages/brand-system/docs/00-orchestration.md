## 0. T H E C O N D U C T O R ' S S C O R E

> _"A system without sequence is noise. A sequence without purpose is bureaucracy. A purpose without execution is philosophy."_

This document governs the autonomous execution of a complete brand system. Seven layers, one axiom, infinite applications.

**The Guarantee**: An AGI reading this protocol, its referenced documents, and a valid Brand Seed can execute the entire system without additional human clarification beyond the seed inputs, producing artifacts that convert attention into revenue while honoring consciousness.

**The Brand Seed Requirement**: The system requires `brand-seed.json` (from `templates/brand-seed.template.json`) containing market evidence, founder assets, structured economic constraints, unit economics inputs, and conversion data. `brand-seed.md` is the human-readable mirror. Without this seed, the system must halt and request the missing information. It will never guess. See `templates/brand-seed.template.json` and `templates/brand-seed.template.md` for the complete input contract.

**The Derivation Log Requirement**: Every layer output must include a `derivation_log` that explicitly traces decisions to the axiom, laws, and canonical SECTION_* sources. This is non-negotiable; if a decision cannot be traced, it must be revised or removed.

**The Fundamental Equation**:

```
Axiom → Philosophy → Strategy → Expression → Behavior → Artifact → Revenue
```

But this equation hides a deeper truth. Revenue is not the goal. Revenue is the **evidence** — proof that attention was earned, consciousness was honored, and value was delivered.

**The Real Equation**:

```
Attention (Sacred) → Trust (Earned) → Transformation (Delivered) → Revenue (Evidence)
```

### The Biological Foundation

Before philosophy, before strategy, before a single pixel is placed — there is **biology**.

The nervous system speaks through **eight hardwired imperatives** — the Life-Force 8 (LF8). These operate beneath conscious thought, driving 80% of human decisions.

> **Canonical Source**: `registry/biological/lf8.json` — All documents must reference LF8 by semantic ID, not rank number.

| Rank | LF8 ID            | Driver                       | The Body's Demand   | Overrides         |
| ---- | ----------------- | ---------------------------- | ------------------- | ----------------- |
| 1    | `LF8_SURVIVAL`    | **Survival**                 | "Keep me alive"     | All others        |
| 2    | `LF8_PAIN_FREEDOM`| **Freedom from Pain**        | "Remove my threats" | Most others       |
| 3    | `LF8_PROTECTION`  | **Protection of Loved Ones** | "Guard my tribe"    | Personal comfort  |
| 4    | `LF8_COMPANIONSHIP`| **Sexual Companionship**    | "Help me connect"   | Status, comfort   |
| 5    | `LF8_COMFORT`     | **Comfortable Living**       | "Make it easy"      | Status, approval  |
| 6    | `LF8_NOURISHMENT` | **Nourishment**              | "Give me pleasure"  | Status, approval  |
| 7    | `LF8_SUPERIORITY` | **Superiority**              | "Help me win"       | Approval          |
| 8    | `LF8_APPROVAL`    | **Social Approval**          | "Help me belong"    | None              |

**The Master Override**: When `LF8_SURVIVAL` is genuinely at stake — user safety, physical harm, legal exposure — all other principles yield. This is not a choice. It is biology.

<a id="SECTION_ORCH_THE_AWARENESS_SPECTRUM"></a>
### The Awareness Spectrum

Before a word is written, before a color is chosen, determine where the prospect stands:

| Level              | State                   | System Response                 |
| ------------------ | ----------------------- | ------------------------------- |
| **Unaware**        | No knowledge of problem | Story lead; pain naming         |
| **Problem-Aware**  | Feels pain, no solution | Agitate; present answer         |
| **Solution-Aware** | Knows result needed     | Connect need to your offer      |
| **Product-Aware**  | Knows product, unsure   | Demonstrate superiority         |
| **Most-Aware**     | Knows product, wants it | Direct offer; compelling reason |

Every layer adapts to this spectrum. Voice shifts. Visuals shift. Moments shift.

<a id="SECTION_ORCH_THE_VOICE_MATRIX"></a>
### The Voice Matrix

Different sections require different voices. The archetype that opens a story differs from the voice that delivers technical specification. Map section types to their appropriate character:

| Section Type | Lead Archetype | Voice Character | Example Use |
|--------------|----------------|-----------------|-------------|
| **Entry/Gate** | Proclamation | Bold, pattern-interrupting | docs/00-gate.md opening |
| **Philosophy** | Story | Narrative, immersive | docs/01-manifesto.md |
| **Diagnosis** | Problem-Solution | Pain → relief, consequence → solution | Awareness sections |
| **Strategy** | Problem-Solution | Pain → relief, curiosity → instruction | docs/02-covenant.md |
| **Technical Spec** | Reporter | Precise, authoritative, minimal | Token Canon, schemas |
| **Decision Protocol** | Teacher | Structured, systematic | Tradeoff Ladder, Decision Protocol |
| **Behavioral** | Instructor | Prescriptive, clear, practical | docs/06-choreography.md |
| **Revenue/Conversion** | Offer | Direct, urgent, value-focused | docs/07-storefront.md |

**Voice Selection Protocol:**

1. Identify the section's primary function (entry, philosophy, strategy, specification, instruction, conversion)
2. Select the matching lead archetype
3. Write in that character's voice
4. Validate: Does the voice match the reader's awareness level at this point in the system?

**The Rule:** A section can shift voice mid-stream, but the opening must match the archetype. The opening earns the right to teach. Teaching earns the right to prescribe. Prescription earns the right to convert.

<a id="SECTION_ORCH_THE_PARADOX"></a>
### The Paradox

Every layer must hold two truths without compromise:

**Completely Logical** — Every choice traces to first principles. Nothing arbitrary. The rational mind finds no loose screws.

**Deeply Mysterious** — The experience exceeds explanation. Emotional response arrives from what is withheld. The whole remains larger than analysis.

```
Logic determines WHAT is present.
Mystery emerges from WHAT is absent.
```

**Autonomous Proxy**: When no human evaluator is available, require the Storefront's Mystery Proxy Audit metrics (whitespace ratio, information density, open-loop ratio, explanation-to-implication ratio) as a minimum machine-check.

  1. Check if derivable from existing layers

  5. If not derivable → flag specific gap for human input
